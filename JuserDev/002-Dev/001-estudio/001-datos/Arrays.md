### Propiedades Básicas

|**Propiedad**|**Descripción**|
|---|---|
|**length**|Devuelve la cantidad de elementos en el array.|

---

### Métodos para Modificar el Array (Mutables)

Estos métodos cambian el array original directamente.

|**Nombre**|**Uso / Descripción**|
|---|---|
|**push()**|Añade uno o más elementos al **final** del array y devuelve la nueva longitud.|
|**pop()**|Elimina el **último** elemento del array y lo devuelve.|
|**shift()**|Elimina el **primer** elemento del array y lo devuelve.|
|**unshift()**|Añade uno o más elementos al **inicio** del array.|
|**splice()**|Cambia el contenido eliminando, reemplazando o agregando elementos en una posición específica.|
|**sort()**|Ordena los elementos del array (por defecto, como strings).|
|**reverse()**|Invierte el orden de los elementos en el array.|

---

### Métodos de Búsqueda y Comprobación

|**Nombre**|**Uso / Descripción**|
|---|---|
|**indexOf()**|Devuelve el primer índice donde se encuentra un elemento; si no existe, devuelve `-1`.|
|**includes()**|Determina si un array contiene un elemento específico (devuelve `true` o `false`).|
|**find()**|Devuelve el **primer elemento** que cumple con una condición lógica.|
|**findIndex()**|Devuelve el **índice** del primer elemento que cumple con la condición.|

---

### Métodos de Transformación (Iteradores)

Estos métodos suelen recorrer el array y ejecutar una función por cada elemento.

|**Nombre**|**Uso / Descripción**|
|---|---|
|**forEach()**|Ejecuta una función para cada elemento (no devuelve nada nuevo).|
|**map()**|Crea un **nuevo array** con los resultados de aplicar una función a cada elemento.|
|**filter()**|Crea un **nuevo array** con todos los elementos que cumplan una condición.|
|**reduce()**|Aplica una función a un acumulador y a cada objeto para reducirlo a un **único valor**.|

---

### Métodos de Combinación y Corte

|**Nombre**|**Uso / Descripción**|
|---|---|
|**slice()**|Devuelve una copia de una parte del array (sin modificar el original).|
|**concat()**|Une dos o más arrays y devuelve uno nuevo.|
|**join()**|Une todos los elementos de un array en una cadena de texto (string), separados por un carácter.|

> **Nota para principiantes:** Si estás trabajando con JavaScript, recuerda que el primer elemento de un array siempre está en la posición `0`, no en la `1`.
