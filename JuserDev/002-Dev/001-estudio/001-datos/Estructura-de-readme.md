## 💡 Acerca del Proyecto

Nací de la frustración de usar complejas herramientas GUI para la gestión de mis propios proyectos de desarrollo. ProjectPulse ofrece una interfaz minimalista y rápida para...

### 🔑 Características

* **Gestión de Tareas:** Crea, edita y elimina tareas desde la terminal.
* **Priorización:** Asigna niveles de prioridad (Baja, Media, Alta).
* **Exportación:** Exporta el estado del proyecto a un archivo JSON.
---
## 💻 Tecnologías Usadas

* **Lenguaje:** Python 3.10+
* **Librerías:** `Click` (para la CLI), `SQLite` (para la base de datos).
* **Dependencias:** `pipenv`
---
## ⚙️ Instalación

### Prerrequisitos

Asegúrate de tener instalado [Python 3.10 o superior](https://www.python.org/downloads/).

### Pasos

1.  Clona el repositorio:
    ```bash
    git clone [https://github.com/tu-usuario/ProjectPulse.git](https://github.com/tu-usuario/ProjectPulse.git)
    ```
2.  Navega al directorio:
    ```bash
    cd ProjectPulse
    ```
3.  Instala las dependencias (usando `pip`):
    ```bash
    pip install -r requirements.txt
    ```
4.  Ejecuta la herramienta:
    ```bash
    python projectpulse.py --help
    ```

---
## 🚀 Uso

Para crear una nueva tarea, usa el comando `add`:

```bash
python projectpulse.py add "Configurar README.md" --prioridad Alta
```
